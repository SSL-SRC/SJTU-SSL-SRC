# FREQ拨码开关与对应频率
小车上FREQ的拨码开关用于设置通信频率，可以设置为0~15。所设置的频率并不是随着拨码开关数值递增的，具体对应关系如下表。如果两台发射机一起用的话**避免使用相邻频道（指频率而不是拨码开关数值）**，防止相邻频道干扰。

| FREQ数值 | 对应频率 |
| -------- | -------- |
|    0     | 2.495GHz |
|    1     | 2.473GHz |
|    2     | 2.475GHz |
|    3     | 2.477GHz |
|    4     | 2.479GHz |
|    5     | 2.481GHz |
|    6     | 2.485GHz |
|    7     | 2.489GHz |
|    8     | 2.501GHz |
|    9     | 2.503GHz |
|    10    | 2.493GHz |
|    11    | 2.505GHz |
|    12    | 2.509GHz |
|    13    | 2.513GHz |
|    14    | 2.517GHz |
|    15    | 2.521GHz |
